类的数据成员不能在(声明!!!)类时初始化
构造函数：
class Time
{
  public:
	Time()         /////
	{
	  hour=0;      构造函数
	  minute =0;
	}               ///////
  private:
	int hour;
	int minute;
};

类外定义构造函数：

Time::Time()
{
	hour=0;
	minute=0;
}

构造函数主要是用来对对象进行初始化。


带参数: 
	Box::Box(int h,int w)
	{
	 height = h;
	 hour = w;
	}
初始化：
	Box::Box(int h,int w):height(h),width(w){}
